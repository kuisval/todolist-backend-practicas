# Aprendiendo Node.js + Full Stack Todo List 
 
Acá voy a documentar mi proceso aprendiendo **Node.js** como entorno de ejecución, y todo lo que vaya construyendo a partir de eso.
 
## ¿Qué hay en este repo?
 
La idea es simple: venía trabajando con JavaScript en el frontend y de backend lo manejaba con supabase y quería ver hasta dónde podía llegar usando el mismo lenguaje pero ahora del lado del servidor. Node.js me permite hacer exactamente eso, ejecutar JS fuera del navegador y empezar a construir backends reales.
 
Como proyecto práctico voy a hacer un **Todo List Full Stack**, sí, el clásico, pero con todo:
 
- Backend con **Node.js + Express** (API REST)
- Conexión a base de datos
- Frontend consumiendo la API
- Deploy funcionando
Nada de tutoriales a medias, esto corre.
 
## ¿Por qué esto?
 
Porque me interesa demostrar que adaptarme a nuevas tecnologías no me cuesta. Ya manejaba JavaScript en el frontend, y ahora estoy tomando ese mismo conocimiento y expandiéndolo al backend. El lenguaje es el mismo, el mindset también.
 Si mañana el stack cambia, cambio con él. Esa es la idea, dejar en claro que me puedo ADAPTAR a todo y con rapidez.
